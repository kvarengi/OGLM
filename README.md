    –†–µ–ø—É—Ç–∞—Ü–∏—è –æ–±–Ω–æ–≤–ª—è–µ—Ç—Å—è –ø–æ —Ä–µ–∑—É–ª—å—Ç–∞—Ç–∞–º –ø—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–∏–π

    prior = node.reputation

    days_inactive = (datetime.now() - node.last_activity).days

        "timestamp": datetime.now(),
